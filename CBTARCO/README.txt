# PROJETO CBTARCO
Este é meu segundo projeto do meu portfolio, ele contem os rankings anuais de 2010 a 2018 do capeonato de Tiro com Arco em Outdoor brasileiro. O projeto está em fase de andamento e começou pelo ano de 2018 que seguirá em ordem decrescente.
Para mais informações procurem a documentação na pasta de cada ano do projeto.
